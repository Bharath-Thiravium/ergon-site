# User Manual: Task Management

## 9.1 Overview
The Task module helps teams collaborate by tracking work items, deadlines, and progress.

## 9.2 Viewing Tasks
1. Navigate to **Tasks** in the main menu.
2. Use the tabs to filter:
   - **My Tasks:** Assigned specifically to you.
   - **Created by Me:** Tasks you assigned to others.
   - **All Tasks:** Team-wide tasks (if authorized).

## 9.3 Creating a New Task
**Procedure:**
1. Click the **Create Task** button.
2. Fill in the details:
   - **Title:** Short summary of the work.
   - **Description:** Detailed instructions.
   - **Assignee:** Select the user responsible.
   - **Priority:** Low, Medium, High, Urgent.
   - **Due Date:** Deadline for completion.
3. (Optional) Attach relevant files.
4. Click **Save Task**.

## 9.4 Updating Task Progress
To update a task you are working on:

1. Click on the Task Title to open the detail view.
2. **Change Status:**
   - Move from *To Do* &rarr; *In Progress* &rarr; *Completed*.
3. **Add Comments:**
   - Type updates or questions in the comment box below.
   - Click **Post Comment**.

## 9.5 Closing a Task
Once the work is done and verified:
1. Change the status to **Completed**.
2. The task will move to the "Completed" archive view.
