# User Manual: Introduction

## 1.1 Welcome to Ergon Site

Ergon Site is a comprehensive Human Resource Management System (HRMS) designed to streamline employee-related processes, enhance productivity, and facilitate better communication within your organization. This manual provides detailed guidance on using various modules of the system.

## 1.2 Key Features

- **Employee Self-Service Portal:** Employees can manage their profiles, submit leave requests, track attendance, and handle financial requests independently.
- **Task and Project Management:** Collaborate on tasks, assign responsibilities, and monitor progress with built-in workflow tools.
- **Financial Management:** Process salary advances, expense claims, and receipt submissions with approval workflows.
- **Attendance Tracking:** Automated or manual attendance recording with GPS validation and regularization options.
- **Administrative Maintenance:** Tools for HR administrators to manage users, roles, and system configurations.
- **File Upload and Document Management:** Secure upload and storage of documents and receipts.

## 1.3 System Architecture

The system is built using PHP, MySQL, and modern web technologies, ensuring scalability and security. It supports multi-tenant configurations for different organizations.

## 1.4 Getting Started

1. **Access the System:** Open your web browser and navigate to the Ergon Site URL provided by your administrator.
2. **Login:** Enter your username and password. New users may need to reset their password on first login.
3. **Dashboard Overview:** Upon login, you'll see a personalized dashboard with widgets for recent activities, pending tasks, and quick links to modules.
4. **Navigation:** Use the main menu to access different modules. The interface is responsive and works on desktop and mobile devices.

## 1.5 User Roles and Permissions

- **Employee:** Basic access to self-service features.
- **Manager:** Additional permissions for approving requests and managing team tasks.
- **HR/Admin:** Full access to user management, reports, and system settings.

## 1.6 System Requirements

- **Browser:** Chrome 90+, Firefox 88+, Edge 90+, Safari 14+
- **Internet Connection:** Stable broadband for optimal performance
- **Device:** Desktop, laptop, or mobile device
- **Additional Software:** None required; all functionality is web-based

## 1.7 Security and Privacy

Ergon Site employs industry-standard security measures including data encryption, secure authentication, and regular backups. All user data is handled in compliance with privacy regulations.

## 1.8 Support and Training

- **Help Documentation:** Access this manual anytime from the system.
- **In-App Support:** Use the help button for context-sensitive assistance.
- **Contact Support:** Reach out to your organization's IT or HR support team for technical issues.
- **Training:** New users are encouraged to complete the onboarding tutorial available in the system.

## 1.10 Frequently Asked Questions (FAQ)

### General Questions
**Q: How do I access Ergon Site?**  
A: Use the URL provided by your administrator and log in with your credentials.

**Q: What if I forget my password?**  
A: Use the "Forgot Password" link on the login page to reset it.

**Q: Is the system mobile-friendly?**  
A: Yes, the interface is responsive and works on all devices.

### Technical Questions
**Q: What browsers are supported?**  
A: Chrome, Firefox, Edge, and Safari (latest versions recommended).

**Q: Do I need to install any software?**  
A: No, it's entirely web-based.

**Q: How secure is my data?**  
A: Data is encrypted and stored securely with regular backups.

## 1.11 Glossary of Terms

- **HRMS:** Human Resource Management System
- **Self-Service Portal:** Area where employees manage their own data
- **Workflow:** Automated process for approvals and tasks
- **Multi-Tenant:** System supporting multiple organizations
- **GPS Validation:** Location verification for attendance
- **Regularization:** Process to correct attendance records
